💻 Part1：建站准备篇

第一步：购买你的网站地基！VPS推荐：

  • CloudCone官网（性价比之王）
    注意：打开官网时建议开启魔法上网速度更快，最低配置建议选2核2G，有条件的上4核4G性能飞起，系统选Ubuntu22.04版本！
  • 亿速云官网（国内线路优化）
    注意：它的香港免备案服务器采用CN2高速线路回国，低至5ms延迟！

第二步：给你的网站起个响亮的名字！Namesilo官网注册教程：

  • 搜索心仪域名
  • 加入购物车时记得勾选WHOIS隐私保护
  • 支付宝付款超方便！

💻 Part2：实战操作篇

第三步：让域名找到你的服务器！Cloudflare解析全流程：

  • 添加站点
  • 修改域名服务器Nameserver
  • 创建A记录指向VPS的IP
  • 小技巧：免费开启CDN加速！”

第四步：武装你的工具箱！必装神器清单：

  • Xshell（SSH连接）
  • 宝塔面板（可视化操作）
  • 性能调优脚本代码：
curl -sS -O https://raw.githubusercontent.com/kejilion/sh/main/kejilion.sh && chmod +x kejilion.sh && ./kejilion.sh

第五步：系统调优四连击！

  • 系统更新： 
    sudo apt-get update && sudo apt-get upgrade -y
  • 虚拟内存设置，运行第四步调优脚本代码，选择13→12:
  • BBR3加速开启，运行第四步调优脚本代码，选13→16:
  • 安全防护，安装fail2ban，运行第四步调优脚本代码，选13→22:

💻 Part3：建站教程终极目标达成

第六步：可视化神器登场！安装如下宝塔面板命令，或前往宝塔官网复制

wget -O install_panel.sh https://download.bt.cn/install/install_panel.sh && sudo bash install_panel.sh ed8484bec

安装完记得保存面板地址/账号密码！”

第七步：安装wordpress，导入主题，网站终成！

  • 宝塔创建网站+数据库
  • 下载wordpress安装包，上传网站根目录
  • 经典5分钟安装流程，见证奇迹的时刻！你的网站诞生了！

恭喜你成为站长！
